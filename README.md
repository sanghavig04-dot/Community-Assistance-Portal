# 🌐 Community Assistance Portal

## 📌 Overview

A full-stack web application designed to help communities connect, request help, and provide assistance efficiently.

## 🚀 Features

* User registration & login
* Request help / provide help
* Dashboard for managing requests
* Real-time updates (if applicable)

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Node.js + Express
* Database: MongoDB

## 📁 Project Structure

```
backend/   → APIs and server logic  
frontend/  → User interface  
```

## ⚙️ Installation

### Backend

```
cd backend
npm install
npm start
```

### Frontend

```
cd frontend
npm install
npm start
```
